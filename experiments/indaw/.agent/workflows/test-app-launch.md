# INDAW 起動テストチェックリスト

## 1. アプリ起動
```powershell
npm run tauri dev
```
起動後、ウィンドウが表示されることを確認。

## 2. レイアウト確認
- [ ] 5ゾーンレイアウト (Top, Bottom, Left, Right, Center) が正しく表示される
- [ ] 分割ペイン (SplitPane) がドラッグでサイズ変更可能
- [ ] TrackList が左側に表示される

## 3. トランスポート (BottomBar)
- [ ] 再生ボタン (▶) をクリック → 音が鳴る
- [ ] 停止ボタン (■) をクリック → 再生が止まる
- [ ] LOOPボタン → ループ ON/OFF が切り替わる
- [ ] プレイヘッド位置 (右下の秒数表示) が更新される

## 4. PianoRoll
- [ ] ピアノ鍵盤 (左端) が表示される
- [ ] 初期トラック (Inst 1) のノートが表示される
- [ ] ノートをクリックで選択できる
- [ ] ノートをドラッグで移動できる
- [ ] Delete キーでノート削除

## 5. TrackList
- [ ] トラック一覧が表示される (初期: Inst 1)
- [ ] トラック選択 → PianoRoll がそのトラックを表示
- [ ] M (Mute) ボタン → 赤くなり、トラックがミュート
- [ ] S (Solo) ボタン → 黄色くなり、そのトラックのみ再生
- [ ] Volume スライダー → 音量変更

## 6. TopBar
- [ ] "Export WAV" ボタンが表示される
- [ ] クリック → パス入力ダイアログ

## 7. RightSidebar (ADSR)
- [ ] Attack/Decay/Sustain/Release スライダーが表示される
- [ ] スライダー変更 → 音色が変わる

## 8. タイムライン
- [ ] タイムラインルーラーをクリック → シーク
- [ ] ルーラーをドラッグ → ループ範囲選択
- [ ] ループ範囲がハイライト表示される

## 9. パフォーマンス
- [ ] ウィンドウリサイズ中に音飛びしない
- [ ] 10秒以上再生しても安定

## 問題発生時
1. コンソールエラーを確認: `npm run tauri dev` の出力
2. Rust パニックの場合: `src-tauri/src/` のログを確認
