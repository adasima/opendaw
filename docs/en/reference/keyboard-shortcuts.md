# Keyboard Shortcuts

> A list of keyboard shortcuts to speed up your workflow in OpenDAW.

## Overview
This reference lists the available keyboard shortcuts designed to minimize mouse usage and provide intuitive, speedy control. The currently implemented shortcuts are as follows.

## Transport

| Action | Shortcut | Description |
|:---|:---|:---|
| Play/Stop | `Space` | Toggles project playback between play and stop (pause). This is disabled while typing in text input fields. |

## Notes & Limitations
