# AI Agent

> An interface for communicating with the AI assistant.

## Overview
The AI Agent panel provides a chat interface for communicating with the AI assistant. You can toggle the visibility of this panel, which appears on the right side of the screen, using the toggle button located in the title bar at the top of the screen.

## Basic Usage
1. Click the **AI Agent** toggle button located in the upper right of the title bar to open the AI Agent panel.
2. Enter text in the chat input field (**Ask AI to automate...**) at the bottom of the panel to send instructions to the AI assistant.
