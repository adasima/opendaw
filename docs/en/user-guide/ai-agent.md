# AI Agent

> An interface for communicating with the AI assistant.

## Overview
The AI Agent panel provides a chat interface and status display for communicating with the AI assistant. It is implemented as a resizable panel located on the right side of the screen.

## Basic Usage
1. Locate the panel positioned on the right side of the screen.
2. Check the current status of the AI from the status display within the panel.
