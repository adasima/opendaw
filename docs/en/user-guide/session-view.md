# Session View

> An Ableton Live-like session view feature, ideal for looping and sketching ideas.

## Overview
The Session View provides a non-linear approach to music production. You can arrange clips per track and play them back as a group using horizontal rows called Scenes. Currently, only a skeleton UI grid using dummy data is implemented.

## Basic Usage
### Switching Views
1. Click the **Session** button in the top panel at the top of the screen.
2. The Session View (clip grid) will be displayed. (To return to the original screen, click **Arrangement**).

### Clips and Scenes (Skeleton)
1. In the Session View, a play button is displayed for each row (Scene) (e.g., `▶ Scene 1`).
2. Clip slots are arranged in each track (column).
3. Clicking a colored **Clip** button or the **+** button in an empty slot will allow you to manipulate clips in the future.

## Detailed Settings
Currently, there are no configurable parameters for the Session View.

## Keyboard Shortcuts
In the current version, there are no keyboard shortcuts dedicated specifically to the Session View.

> - In the current version, the Session View is only a UI skeleton. Actual editing of clip data and synchronized playback with the audio engine are not yet supported.
