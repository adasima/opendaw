# Plugin Browser

> A panel for browsing and loading plugins (VST/CLAP, etc.) into tracks.

## Overview
The Plugin Browser provides an interface for loading external plugins (such as VST3 and CLAP) into your project.
By selecting a plugin from the browser, you can add it to the currently active track.

## Basic Usage
1. Click the **Browser** toggle switch on the main screen.
2. The **Plugin Browser** window will appear, displaying a list of available plugins.
3. In the Tracks panel, click on the track you want to add a plugin to, making it active.
4. Click on the desired plugin in the browser, or select it and press the `Enter` key, to load it into the active track.
5. Loaded plugins are displayed in the **Plugins** section of the **Track Details** panel on the right.

## Searching for Plugins
You can filter the plugin list by name or vendor by typing text into the search box at the top of the browser.
