# Automation

> This feature allows you to smoothly change parameters such as volume and pan over time.

## Overview
Automation is a feature that automatically controls track parameters (volume, left-right panning, etc.) as the song plays. It enables expressions such as gradually increasing the volume at a specific time (fade-in) or panning the sound from left to right.

## Basic Usage
### Toggling Automation Visibility
1. In the track list on the main screen (left side), click the **A** (Automation) button located in the header of the track you want to control.
2. When automation is enabled, the **A** button will be highlighted, and the automation controls will appear below the track.
3. Click the **A** button again to turn off the automation display.

### Selecting Parameters
1. With automation enabled, click the displayed drop-down menu (by default, `Volume` is selected).
2. Select the parameter you want to control (such as `Volume` or `Pan`).

## Detailed Settings
Currently, the automation parameters that can be selected from the UI are as follows.

| Parameter Name | Description |
|:---|:---|
| Volume | Controls the volume of the track |
| Pan | Controls the left-right panning of the track |
