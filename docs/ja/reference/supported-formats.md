# 対応フォーマット一覧

> OpenDAW がインポートおよびエクスポートでサポートするオーディオファイルの形式（フォーマット）の一覧です。

## 概要
OpenDAW では、内部のオーディオ処理に高精度な 32-bit Float を採用しています。外部ファイルとのやり取り（インポートおよびエクスポート）においては、標準的な WAV 形式をサポートしています。

## インポート対応フォーマット
オーディオファイルの読み込み（インポート）では、以下の WAV 形式をサポートしています。読み込まれたオーディオデータは、自動的に内部処理用のフォーマット（32-bit Float、-1.0 〜 1.0 に正規化）に変換されます。

| 項目 | サポートされる仕様 | 備考 |
|:---|:---|:---|
| コンテナ形式 | WAV (`.wav`) | 他の形式（MP3, FLAC, OGG等）は未対応 |
| サンプルフォーマット | 整数 (Int) / 浮動小数点 (Float) | |
| ビット深度 (Int) | 8-bit, 16-bit, 24-bit, 32-bit | 8-bit は符号なし、16-bit 以上は符号付き |
| ビット深度 (Float) | 32-bit | 64-bit Float には未対応 |
| チャンネル数 | モノラル (1ch) / ステレオ (2ch) | |
| サンプリングレート | 制限なし | 一般的な 44.1kHz や 48kHz などに対応 |

## エクスポート対応フォーマット
プロジェクトから書き出される（エクスポート）ファイルの形式は、現在以下の仕様に固定されています。

| 項目 | 設定値 |
|:---|:---|
| コンテナ形式 | WAV (`.wav`) |
| チャンネル数 | ステレオ (2ch) |
| サンプルフォーマット | 整数 (Int) |
| ビット深度 | 16-bit |

## 注意事項・制限
> - 現在のバージョンでは、WAV 以外の形式（AIFF、MP3、FLAC など）の読み書きはサポートされていません。
> - エクスポート時のフォーマット変更機能（ビット深度やサンプリングレートの選択）は今後のアップデートで追加予定です。
