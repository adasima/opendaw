# プラグインブラウザ

> プラグイン（VST/CLAP等）を閲覧・ロードするためのパネルです。（現在はモックアップ段階）

## 概要
プラグインブラウザは、外部プラグイン（VST3、CLAPなど）をプロジェクトに読み込むためのインターフェースを提供します。
現在はUIのスケルトンのみが実装されており、実際のプラグインのロードは今後のアップデートで追加される予定です。

## 基本的な使い方
1. メイン画面のトグルスイッチから **Browser** をクリックします。
2. **Plugin Browser** ウィンドウが表示されます。
3. リストに表示されているプラグイン（現在はダミーデータ）を確認します。
4. 各プラグインの横にある **Load** ボタンをクリックします（現在は何も起きません）。

## 注意事項・制限
> 現在のバージョンでは、プラグインブラウザはUIのスケルトンのみが実装されています。
> ダミーデータ（例: "Serum (Dummy)"）が表示されますが、実際のプラグインの読み込みや音声処理は機能しません。
