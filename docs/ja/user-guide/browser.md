# プラグインブラウザ

> プラグイン（VST/CLAP等）を閲覧し、トラックにロードするためのパネルです。

## 概要
プラグインブラウザは、外部プラグイン（VST3、CLAPなど）をプロジェクトに読み込むためのインターフェースを提供します。
ブラウザからプラグインを選択することで、アクティブなトラックにプラグインを追加することができます。

## 基本的な使い方
1. メイン画面のトグルスイッチから **Browser** をクリックします。
2. **Plugin Browser** ウィンドウが表示され、利用可能なプラグインがリスト表示されます。
3. トラックパネルから、プラグインを追加したいトラックをクリックして選択状態（アクティブ）にします。
4. ブラウザ内の目的のプラグインをクリック、または選択して `Enter` キーを押すことで、アクティブなトラックにロードされます。
5. ロードされたプラグインは、右側の **Track Details** パネルの **Plugins** セクションに表示されます。

## プラグインの検索
ブラウザ上部の検索ボックスにテキストを入力することで、プラグイン名やベンダー名でリストをフィルタリングすることができます。
