# トラック管理

> プロジェクト内の各トラック（オーディオ、MIDIなど）の追加・削除や状態を管理する機能です。

## 概要
トラックパネルは画面左側に配置されており、現在のプロジェクトに存在するトラックの一覧を表示します。トラックは独立したオーディオソースやインストゥルメントを割り当てるための基本的な単位です。

## 基本的な使い方
1. パネル上部の **+ Add Track** ボタンをクリックして、新しいトラックを追加します。
2. トラック一覧の各行にある **X** ボタンをクリックして、そのトラックを削除します。

## 詳細設定
各トラックは内部的に以下のパラメータを持っています。（※現在のUIから直接操作できる項目は限られています）

| パラメータ | 説明 | 初期値 | 範囲 |
|:---|:---|:---|:---|
| ID | トラックの一意な識別子（自動生成） | - | 1〜 |
| Name | トラックの名前 | "Track X" | - |
| Volume | トラック個別の出力音量 | 1.0 | 0.0 〜 無制限 (ブースト可) |
| Pan | パンニング（左右の定位） | 0.0 | -1.0 (左) 〜 1.0 (右) |
| Mute | トラック単位の消音状態 | Off | On / Off |
| Solo | ソロ状態。Onの場合、SoloがOnのトラックだけが再生されます | Off | On / Off |

## キーボードショートカット
現在のバージョンでは、トラック操作専用のキーボードショートカットはありません。

## 注意事項・制限
> 現在のバージョンでは、トラックの追加および削除のみUIから操作可能です。ボリューム、パン、ミュート、ソロの個別設定、およびトラック名の変更は内部状態として定義されていますが、UIには未実装です。
