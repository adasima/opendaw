# トラック管理

> プロジェクト内の各トラック（オーディオ、MIDI、シンセサイザーなど）の追加や選択、詳細設定を行う機能です。

## 概要
トラックパネルは画面左側に配置されており、現在のプロジェクトに存在するトラックの一覧を表示します。トラックは独立したオーディオソースやインストゥルメントを割り当てるための基本的な単位です。トラックを選択することで、右側に **Track Details**（トラック詳細）パネルが開き、より高度な設定を行うことができます。

## 基本的な使い方
### トラックの追加と選択
1. パネル上部の **＋** ボタンをクリックして、新しいトラックを追加します。（将来的にトラックの種類の選択が追加される予定です）
2. トラック一覧の任意のトラックをクリックすると、そのトラックがアクティブ（選択状態）になり、右側に **Track Details** パネルが表示されます。

### トラック上のコントロール
トラックヘッダーには以下のボタンが配置されており、素早く状態を切り替えることができます。

- **M** (Mute): このトラックの音をミュート（消音）します。
- **S** (Solo): このトラックをソロ状態にします。ソロが有効なトラックのみが再生されます。
- **R** (Record): このトラックの録音待機（レコードアーム）状態を切り替えます。

## Track Details（詳細設定）
トラックを選択すると表示される **Track Details** パネルでは、以下の設定が可能です。

### Mixer
| パラメータ | 説明 | 初期値 | 範囲 |
|:---|:---|:---|:---|
| Volume | トラック個別の出力音量 | 0.8 (80%) | 0.0 〜 1.0 (0%〜100%) |
| Pan | パンニング（左右の定位） | 0.0 (0%) | -1.0 (左) 〜 1.0 (右) |

### MIDI Routing
| パラメータ | 説明 |
|:---|:---|
| Input Device | MIDI信号を受信する入力デバイスを選択します |
| Channel | 受信するMIDIチャンネル（0=All, 1〜16）を指定します |

## キーボードショートカット
現在のバージョンでは、トラック操作専用のキーボードショートカットはありませんが、トラックを選択した状態で `Enter` キーを押すことでアクティブにすることができます。

## 注意事項・制限
> - 現在のバージョンでは、UI上からのトラックの削除（Xボタン等）はサポートされていません。
> - MIDI Routingの設定は、接続されているMIDIデバイスがある場合のみ有効に機能します。
