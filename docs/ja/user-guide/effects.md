# エフェクト

> 各トラックに対して、音量調整（ゲイン）や音質調整（フィルター）などのエフェクトを追加・管理する機能です。

## 概要
エフェクトチェーンを使用して、オーディオトラックの音を加工します。現在、Gain（ゲイン）とFilter（フィルター）の2種類のエフェクトがサポートされており、トラックごとに複数のエフェクトを組み合わせて適用することができます。

## 基本的な使い方
1. メイン画面のトラック一覧から、エフェクトを適用したいトラックの **Effects** ボタンをクリックします。
2. 開いたエフェクトウィンドウ上部の **Add Gain** または **Add Filter** ボタンをクリックして、エフェクトを追加します。
3. 追加されたエフェクトは、リスト形式で表示され、上から順番にオーディオ信号に適用されます。

## エフェクトの管理
追加した各エフェクトに対して、以下の操作が可能です。

| 操作 | 説明 |
|:---|:---|
| 有効/無効の切り替え | 各エフェクトの左側にあるチェックボックスをクリックすると、そのエフェクトの有効/無効（バイパス）を切り替えることができます。 |
| 順序の変更 | エフェクト右側の **↑** または **↓** ボタンをクリックして、エフェクトが適用される順番を変更します。 |
| 削除 | エフェクト右側の **X** ボタンをクリックすると、そのエフェクトをトラックから削除します。 |

## 対応エフェクト

現在、以下のエフェクトが利用可能です。

| エフェクト名 | 説明 |
|:---|:---|
| Gain | トラックの音量を調整します。 |
| Filter | 特定の周波数帯域をカットまたは強調するローパスフィルター（Biquad）です。 |

> 現在のバージョンでは、エフェクトの各種パラメータ（Gainの音量値やFilterのカットオフ周波数など）をUIから直接編集する機能は開発中です。
