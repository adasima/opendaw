# プロジェクト管理

> プロジェクトの現在の状態をファイルに保存し、後から読み込んで作業を再開する機能です。

## 概要
作成した楽曲のBPMやマスターボリューム、追加したトラックなどの状態を `.aura` 拡張子のプロジェクトファイルとして保存できます。保存したファイルを開くことで、DAWを再起動した後でも以前の状態から編集を再開することが可能です。

## 基本的な使い方
### プロジェクトの保存
1. メニューバーまたはUI上部の **💾 Save Project** ボタンをクリックします。
2. ファイル保存ダイアログが開くので、保存先のフォルダとファイル名を入力します。
3. 保存を実行すると、`.aura` 形式のファイルとしてプロジェクトが保存されます。

### プロジェクトの読み込み
1. メニューバーまたはUI上部の **📂 Load Project** ボタンをクリックします。
2. ファイル選択ダイアログが開くので、以前に保存した `.aura` ファイルを選択します。
3. ファイルが読み込まれ、DAWの状態（BPMやトラック構成など）が復元されます。

## 詳細設定
現在、プロジェクトファイル（`.aura`）には以下の情報が含まれます。

| 項目 | 説明 |
|:---|:---|
| BPM | プロジェクトのテンポ（Beats Per Minute） |
| マスターボリューム | 全体の出力音量 |
| トラック情報 | 追加されている各トラックの名前や設定 |
| グリッド設定 | グリッドの有効/無効状態および解像度（1/4など） |

※再生状態（再生中/停止中）やプレイヘッドの位置など、一時的な再生状態は保存されず、読み込み時は初期状態にリセットされます。

## キーボードショートカット
現在のバージョンでは、プロジェクトの保存・読み込み専用のキーボードショートカットはありません。

## 注意事項・制限
> - 読み込み時にエラーが発生した場合、プロジェクトの状態は更新されず、エラーログが出力されます。
> - 他のPCにプロジェクトファイルを移動して開く場合、プロジェクト内で参照されているオーディオファイル（インポートしたWAV等）への絶対パスが変わると正しく再生されない可能性があります。
