# クリップ (Clips)

> オーディオデータやMIDIデータを保持する単位「クリップ」に関する機能です。

## 概要
クリップは、トラック上に配置されるオーディオデータやMIDIシーケンスの実体です。現在のプロジェクトでは、録音された波形データを持つ「オーディオクリップ」と、ピアノロールで編集可能なノートデータを持つ「MIDIクリップ」の2種類が存在します。

## オーディオクリップ (AudioClip)
録音された音声データやインポートされたオーディオファイルを表します。
- **ID**: クリップの一意な識別子
- **Name**: クリップ名（例: "Recorded Clip"）
- **Start Position**: タイムライン上の開始位置（秒単位など）
- **Length**: クリップの長さ（秒単位など）
- **Waveform Summary**: 波形描画用のサマリーデータ

## MIDIクリップ (MidiClip)
ピアノロールエディタで作成・編集されるMIDIノートの集合を表します。
- **ID**: クリップの一意な識別子
- **Name**: クリップ名（例: "Synth Melody"）
- **Start Beat**: タイムライン上の開始位置（拍単位など）
- **Length**: クリップの長さ（拍単位など）
- **Sequence**: ノートの配置や長さを保持するシーケンスデータ
