# エクスポート

> プロジェクトのミキサー出力をWAVファイルとして書き出す（オフラインレンダリング）機能です。

## 概要
エクスポート機能を使用することで、作成した楽曲やオーディオプロジェクトを1つのWAVファイル（16ビット、ステレオ）として書き出すことができます。これにより、他のプレイヤーでの再生や共有が可能になります。

## 基本的な使い方
1. （※現在UIからのエクスポート操作は未実装です。`src/engine/export.rs` にてバックエンド機能として実装されています。）

## 詳細設定
現在のバージョンでは、エクスポート時のフォーマットは以下のように固定されています。

| パラメータ | 設定値 |
|:---|:---|
| チャンネル数 | 2 (ステレオ) |
| ビット深度 | 16ビット |
| サンプルフォーマット | 整数 (Int) |
| サンプリングレート | プロジェクトの設定に依存 |

## 注意事項・制限
> - 現在のバージョンでは、UIからエクスポートを実行するボタンやメニューは実装されていません。機能自体はエンジン内部(`src/engine/export.rs`)に組み込まれています。
> - エクスポートされるファイル形式は `.wav` のみに対応しています。
