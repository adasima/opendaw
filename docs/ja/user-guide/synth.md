# シンセサイザー (Software Instrument)

> プロジェクト内で音を生成する内蔵ソフトウェアインストゥルメント（シンセサイザー）の機能です。

## 概要
シンセサイザー機能は、DAW内部でオーディオ波形を生成し、発音する機能です。基本的なオシレータとADSRエンベロープを備えており、外部のオーディオファイルをインポートしなくても、DAW単体で音作りを始めることができます。現在はサイン波、矩形波、ノコギリ波の3種類の波形をサポートしています。

## 基本的な使い方
### シンセサイザートラックの追加
1. 画面左側のトラックパネル上部にある **+ Add Synth Track** ボタンをクリックします。
2. シンセサイザーが有効化された新しいトラック（例: "Synth 1"）が追加されます。

### 音量の調整とパラメータの変更
1. 画面下部のミキサーパネルで、追加したトラックのコントロールを探します。
2. **Volume** スライダーで音量を調整します。
3. **Freq (Hz)** スライダーを左右に動かして、生成される音の高さを変更します。

## 詳細設定
各シンセサイザーは以下のパラメータを持っています。これらの設定はプロジェクトファイル（`.aura`）に保存されます。

| パラメータ | 説明 | 初期値 | 範囲 |
|:---|:---|:---|:---|
| Synth | シンセサイザーの有効/無効状態 | Off (Synthトラック追加時はOn) | On / Off |
| Freq (Hz) | 生成する基本周波数 | 440.0 | 20.0 〜 20000.0 |
| Waveform | オシレーターの波形 (Sine, Square, Sawtooth) | Sine | - |
| Attack | ADSR: アタックタイム（秒） | 0.01 | 0.0 〜 (上限なし) |
| Decay | ADSR: ディケイタイム（秒） | 0.1 | 0.0 〜 (上限なし) |
| Sustain | ADSR: サステインレベル | 0.5 | 0.0 〜 1.0 |
| Release | ADSR: リリースタイム（秒） | 0.1 | 0.0 〜 (上限なし) |

## キーボードショートカット
現在のバージョンでは、シンセサイザー操作専用のキーボードショートカットはありません。

## 注意事項・制限
> - 現在のバージョンでは、波形(Waveform)の選択およびADSRエンベロープの各パラメータは内部エンジンには実装されていますが、UI上のコントロールが未実装のため、画面上からは周波数(Freq)のみ変更可能です。これらのUIは今後のアップデートで追加予定です。
> - 現在のバージョンでは、シンセサイザーはアクティブなMIDIノート（ピアノロールからの入力など）を受信した際に発音します。MIDIノートがない場合は無音になります。
