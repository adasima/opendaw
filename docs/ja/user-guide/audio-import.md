# オーディオファイルのインポート

> ローカルファイルシステムからオーディオファイル（WAV形式）をプロジェクトに読み込む機能です。

## 概要
オーディオインポート機能を使用することで、外部で作成されたオーディオ素材や録音したデータを現在のプロジェクトに取り込むことができます。現在インポートされたファイルは新しいトラックとして自動的に追加されます。

## 基本的な使い方
1. メイン画面のUI上に配置されている **📁 Import Audio** ボタンをクリックします。
2. ファイル選択ダイアログが開きます。
3. 読み込みたい `.wav` ファイルを選択し、「開く」をクリックします。
4. ファイルの拡張子を除いたファイル名を持つ新しいトラックが、トラック一覧に追加されます。

## 詳細設定
現在のバージョンでは、インポート時に設定可能なパラメータはありません。

| パラメータ | 説明 | 初期値 | 範囲 |
|:---|:---|:---|:---|
| なし | なし | なし | なし |

## 注意事項・制限
> - 現在対応しているオーディオフォーマットは **WAVファイル** のみです。
> - 現行の実装（Phase 3）では、ファイルの読み込み処理（オーディオエンジンへの送信）は未完了であり、トラックの追加のみが実施されます。実際の音声データの再生については今後のアップデートでサポートされる予定です。
