# 録音 (Recording)

> マイクや外部入力からのオーディオ録音を行う機能です。

## 概要
OpenDAWは、システムに接続されたマイクやオーディオインターフェースからの入力を録音し、タイムライン上にオーディオクリップとして配置することができます。録音された波形はリアルタイム（録音停止時）に描画され、視覚的に確認できます。

## 基本的な使い方
1. メイン画面中央下のトランスポートパネルにある **⏺ (Record)** ボタンをクリックします。
2. 録音モードが有効になり、ボタンが赤色の **⏺ (On)** に変わります。
3. トランスポートパネルの再生ボタン (**▶**) をクリックすると、再生と同時に録音が開始されます。
4. 録音を終了するには、再度 **⏺** ボタンをクリックして **⏺ (Off)** にするか、再生を停止 (**⏹** または **⏸**) します。
5. 録音停止時、プロジェクトの最初のトラック（トラックがない場合は自動的に作成されます）に、録音されたオーディオクリップが追加され、波形が表示されます。

## 詳細設定
現在、録音機能に関する詳細な設定画面（デバイスの選択やバッファサイズなど）はUI上に実装されていません。
録音はシステムのデフォルト入力デバイスを自動的に使用します。

| パラメータ | 説明 | 初期値 |
|:---|:---|:---|
| 録音デバイス | 使用するオーディオ入力デバイス | システムのデフォルト |
| サンプルフォーマット | 内部で処理されるオーディオフォーマット | 32-bit Float または 16-bit Int |

## キーボードショートカット
現在のバージョンでは、録音操作専用のキーボードショートカットはありません。

## 注意事項・制限
> - 現在のバージョンでは、入力デバイスをUIから手動で選択することはできません。システムのデフォルト入力デバイスが使用されます。
> - 録音されたクリップは常に最初のトラック（トラックID 0）に追加されます。任意のトラックへのルーティング設定は今後のアップデートで対応予定です。
