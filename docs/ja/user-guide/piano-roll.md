# ピアノロールエディタ

> マウス操作で直感的にMIDIノートを入力・編集できる、内蔵シーケンサーインターフェースです。

## 概要
ピアノロールエディタは、楽曲のメロディやコード、リズムパターンを視覚的に作成・編集するための機能です。グリッドベースの画面上で、ノート（音符）の配置や長さをグラフィカルに調整できます。ARA2 / SV2由来の歌詞表示やピッチベンドカーブの表示にも対応しています。

## 基本的な使い方
1. **ノートの追加**: ピアノロール上の任意のグリッドを「左クリック」します（デフォルトで16分音符が追加されます）。
2. **ノートの削除**: 削除したい既存のノート上で「右クリック」します。
3. **ノートの移動**: ノートの中央をドラッグして、位置（時間）やピッチ（音階）を変更します。
4. **ノートの長さ変更**: ノートの右端をドラッグして長さを変更します。

## 画面の操作
1. **スクロール (パンニング)**: マウスホイール、または「中クリック」しながらドラッグします。
2. **ズーム**: `Ctrl` + マウスホイールで、水平方向のズームイン・ズームアウトを行います。

## 詳細設定
現在のバージョンでは、ピアノロール画面は以下の仕様に基づき動作します。

| パラメータ | 説明 | 初期値 | 範囲 |
|:---|:---|:---|:---|
| Pitch | 音階 (MIDIノート番号) | C-1 〜 G9 | 0 〜 127 |
| Velocity | 音の強さ | 100 | 固定 |
| Duration | ノートの長さ | 16分音符 | 変更可能 |

## 注意事項・制限
> - 現在、ノートの追加時はベロシティ (100) が固定値として設定されます。

## グリッドスナップ
ピアノロール上でノートの追加、移動、長さの変更を行う際、トランスポートパネルでグリッド設定が有効な場合は、選択された解像度に合わせて操作がスナップ（吸着）されます。
