# ピアノロールエディタ

> マウス操作で直感的にMIDIノートを入力・編集できる、内蔵シーケンサーインターフェースです。

## 概要
ピアノロールエディタは、楽曲のメロディやコード、リズムパターンを視覚的に作成・編集するための機能です。グリッドベースの画面上で、ノート（音符）の配置や長さをグラフィカルに調整できます。

## 基本的な使い方
1. **ノートの追加**: ピアノロール上の任意のグリッドを「左クリック」します。
2. **ノートの移動**: 配置されたノートを「左クリック」したままドラッグし、位置やピッチを変更します。
3. **ノートの削除**: 削除したい既存のノート上で「右クリック」します。
4. **スクロール・パンニング**: 画面上をホバーした状態でマウスホイールをスクロールするか、「中クリック」しながらドラッグすることで表示範囲を移動できます。
5. **ズーム**: `Ctrl` キーを押しながらマウスホイールをスクロールすることで、時間軸のズームイン・ズームアウトが可能です。

## 詳細設定
現在のバージョンでは、ピアノロール画面は以下の仕様に基づき動作します。

| パラメータ | 説明 | 初期値 | 範囲 |
|:---|:---|:---|:---|
| Pitch | 音階 (MIDIノート番号) | C4 付近 (起動時の中心表示) | 0 〜 127 |
| Velocity | 音の強さ | なし (未実装) | 固定 |
| Duration | ノートの長さ | 16分音符相当 | 固定 |

## 注意事項・制限
> - 現在、ノートの追加時は長さが16分音符相当として設定されます。
> - ノートの端をドラッグしての長さ変更機能は、今後のアップデートで追加予定です。
> - ピアノロールのノートのベロシティは現在未実装です。
