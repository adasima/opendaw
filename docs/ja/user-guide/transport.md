# トランスポートコントロール

> 楽曲の再生、停止、録音、ループ、テンポ（BPM）などを制御する機能です。

## 概要
トランスポートコントロールは、タイムライン上の再生位置（プレイヘッド）を操作し、プロジェクトの再生状態を管理するために必要です。画面中央下のメインパネルに配置されています。

## 基本的な使い方
1. **⏺ (Record)** ボタンをクリックして録音モードを切り替えます。録音中はボタンが赤色の **⏺ (On)** に変わります。
2. **▶ (Play)** ボタンをクリックして再生を開始します。再生中はボタンが **⏸ (Pause)** に変わります。
3. **⏸ (Pause)** ボタンをクリックして再生を一時停止します。
4. **⏹ (Stop)** ボタンをクリックすると、再生を停止しプレイヘッドを先頭（0.0）に戻します。
5. **🔁 (Loop)** ボタンをクリックして、ループ再生のオン/オフを切り替えます。ループがオンの場合、タイムラインの最後まで再生されると自動的に先頭から再生が再開されます。
6. **⏱ (Metronome)** ボタンをクリックして、メトロノームのオン/オフを切り替えます。

## 詳細設定

| パラメータ | 説明 | 初期値 | 範囲 |
|:---|:---|:---|:---|
| BPM | プロジェクトのテンポ（1分間あたりのビート数）。再生速度に影響します。 | 120.0 | 20.0 〜 300.0 |
| Loop | オンにすると、再生がタイムラインの終端に達した際に先頭に戻って再生を続けます。 | On | On / Off |
| Metronome | メトロノームの有効/無効状態。 | Off | On / Off |
| Recording | 録音の有効/無効状態。 | Off | On / Off |

## キーボードショートカット
現在のバージョンでは、トランスポート操作専用のキーボードショートカットはありません。

## 注意事項・制限
> 現在のバージョンでは、BPMの変更は全体の再生速度に単純に比例して影響します。また、プレイヘッドの位置はパーセンテージ（0.0 〜 100.0）で管理されており、仮想的な秒数（Time）はプレイヘッド位置に基づいて計算・表示されます。

## グリッド設定 (Grid Settings)
タイムラインやピアノロール上でのクリップ・MIDIノートの配置や移動時に、指定した解像度（拍や小節）にスナップ（吸着）させる機能です。

1. **トランスポートパネルのグリッドボタン** をクリックして、グリッドスナップのオン/オフを切り替えます。
2. その隣の **解像度プルダウン**（例: `1/4`, `1/8`, `1/16` など）から、スナップする単位を選択します。
3. グリッドが有効な場合、クリップのドラッグ移動や、ピアノロールでのノートの追加・移動・長さ変更が、選択した解像度に合わせて自動的に補正されます。
