# メトロノーム (Metronome)

> プロジェクトのテンポ（BPM）に合わせてクリック音を生成し、録音や演奏のガイドとなる機能です。

## 概要
メトロノームは、再生位置と設定されたBPMに基づいて定期的なクリック音を発生させます。
1拍目にはアクセント（高音）が鳴り、それ以外の拍では通常のクリック音が鳴ることで、小節の頭を把握しやすくします。

## 基本的な使い方
1. 画面上部のトランスポートパネルにある **⏱** (メトロノーム) ボタンをクリックします。
2. ボタンの表示が **⏱ (On)** に変わり、メトロノームが有効になります。
3. トランスポートパネルの再生ボタン (**▶**) をクリックして再生を開始すると、設定されたBPMに合わせてクリック音が鳴ります。
4. メトロノームを停止するには、再度 **⏱** ボタンをクリックして **⏱ (Off)** にします。

## 詳細設定
メトロノームの内部仕様は以下の通りです。

| パラメータ | 説明 | 設定値 |
|:---|:---|:---|
| 通常クリック周波数 | 2拍目以降のクリック音の高さ | 1000.0 Hz |
| アクセント周波数 | 1拍目（小節の頭）のクリック音の高さ | 1500.0 Hz |
| クリック長 | 1回のクリック音が鳴る時間 | 0.05 秒 |
| 拍子 | 1小節あたりの拍数（現在は固定） | 4 拍 |

## キーボードショートカット
現在のバージョンでは、メトロノーム操作専用のキーボードショートカットはありません。
