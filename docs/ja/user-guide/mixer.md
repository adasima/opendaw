# ミキサー

> プロジェクト全体の音量（マスターボリューム）や各トラック個別のミキシング（音量、パン、ミュート、ソロ）を調整する機能です。

## 概要
ミキサーパネルは、オーディオ出力の最終的な音量レベルや各トラックのバランスを管理するために使用します。画面下部に「Mixer & Effects」パネルとして配置されており、マスターチャンネルおよび各トラックのミキサーコントロール（Volume、Pan、Mute、Solo）を操作可能です。

## 基本的な使い方
### マスターコントロール
1. **Master Volume** スライダーを左右にドラッグして、全体の出力音量を調整します。
2. **🔊** / **🔇** ボタンをクリックして、全体の音声出力のミュート（消音）状態を切り替えます。

### トラックコントロール
トラックごとのパネルでは以下の操作が可能です。
1. **Volume** スライダーを左右にドラッグして、トラックの出力音量を調整します。
2. **Pan** スライダーを左右にドラッグして、トラックのパンニング（左右の定位）を調整します。
3. **M** または **M (On)** ボタンをクリックして、トラックのミュート状態を切り替えます。
4. **S** または **S (On)** ボタンをクリックして、トラックのソロ状態を切り替えます。
5. **FX** ボタンをクリックして、そのトラックのエフェクト設定画面を開きます。
6. **Synth** チェックボックスをクリックして、シンセサイザーの有効/無効を切り替えます。
7. Synthが有効な場合、**Freq (Hz)** スライダーでシンセサイザーの周波数を調整します。

## 詳細設定
ミキサーパネルでは以下のパラメータを操作します。

### マスターチャンネル
| パラメータ | 説明 | 初期値 | 範囲 |
|:---|:---|:---|:---|
| Master Volume | プロジェクト全体の最終的な出力音量 | 0.8 | 0.0 〜 1.0 |
| Mute | 全体の音声を消音するかどうかの状態 | Off (🔊) | On (🔇) / Off (🔊) |

### 各トラック
| パラメータ | 説明 | 初期値 | 範囲 |
|:---|:---|:---|:---|
| Volume | トラックの出力音量 | 1.0 | 0.0 〜 2.0 (ブースト可) |
| Pan | パンニング（左右の定位） | 0.0 | -1.0 (左) 〜 1.0 (右) |
| Mute | トラック単位の消音状態 | Off (M) | On (M (On)) / Off (M) |
| Solo | ソロ状態。Onの場合、SoloがOnのトラックだけが再生されます | Off (S) | On (S (On)) / Off (S) |
| FX | エフェクト設定画面を開く | - | - |
| Synth | シンセサイザーの有効/無効状態 | Off | On / Off |
| Freq (Hz) | シンセサイザーの周波数（SynthがOnの場合のみ） | 440.0 | 20.0 〜 20000.0 |

## キーボードショートカット
現在のバージョンでは、ミキサー操作専用のキーボードショートカットはありません。
