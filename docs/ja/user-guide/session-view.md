# セッションビュー (Session View)

> ループやアイデアのスケッチに最適な、Ableton Liveライクなセッションビュー機能です。

## 概要
セッションビューは、非直線的な音楽制作アプローチを提供するビューモードです。トラックごとにクリップを配置し、シーンと呼ばれる水平方向の行ごとに一括再生することができます。現在はスケルトンUIとして、ダミーデータを用いたグリッド表示のみ実装されています。

## 基本的な使い方
### ビューの切り替え
1. トラックパネル（画面左側）の上部にある **Session** ボタンをクリックします。
2. セッションビュー（クリップグリッド）が表示されます。（元の画面に戻るには **Timeline** をクリックします）

### クリップとシーン（スケルトン）
1. セッションビューでは、行（Scene）ごとに再生ボタンが表示されます（例: `▶ Scene 1`）。このボタンをクリックすると、そのシーンの全クリップが一括して再生状態になります。
2. 各トラック（列）には、クリップスロットが配置されています。
3. 色付きの **Clip** ボタンをクリックすると、そのクリップが再生状態（⏹）になります。
4. 空のスロットの **＋** ボタンをクリックすることで、将来的にクリップの操作が行えるようになります。

## 詳細設定
現在、セッションビューの設定可能なパラメータはありません。

## キーボードショートカット
現在のバージョンでは、セッションビュー専用のキーボードショートカットはありません。

## 注意事項・制限
> - 現在のバージョンでは、セッションビューはUIのスケルトンのみ実装されており、実際のクリップデータの編集やオーディオエンジンの同期再生はサポートされていません。
