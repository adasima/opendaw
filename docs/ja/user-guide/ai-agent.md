# AIエージェント

> AIアシスタントとのコミュニケーションを行うためのインターフェースです。

## 概要
AIエージェントパネルは、AIアシスタントとのコミュニケーション用チャットインターフェースを提供する機能です。画面上部のタイトルバーにあるトグルボタンを使用して、右側に配置されるパネルの表示・非表示を切り替えることができます。

## 基本的な使い方
1. タイトルバーの右上にある **AI Agent** トグルボタンをクリックして、AIエージェントパネルを開きます。
2. パネル下部のチャット入力欄（**Ask AI to automate...**）にテキストを入力し、AIアシスタントに指示を出します。
