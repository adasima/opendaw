# インストール方法

> OpenDAW は数MBの単一バイナリとして提供され、インストール不要で即座に起動できます。

## 概要
OpenDAW の実行環境を準備するためのガイドです。
用途に合わせて、ポータブルバイナリを使用する方法と、ソースコードからビルドする方法を選択できます。

## バイナリで実行する（推奨）
最も手軽に OpenDAW を試す方法です。

1. GitHub リポジトリの [Releases](https://github.com/adasima/opendaw/releases) ページにアクセスします。
2. ご利用のOSに合った最新のZIPファイル（例: `opendaw-portable-windows.zip`）をダウンロードします。
3. ダウンロードしたZIPファイルを任意のフォルダに解凍します。
4. 解凍したフォルダ内の `opendaw` (Windowsの場合は `opendaw.exe`) をダブルクリックして起動します。

## ソースコードからビルドする
最新の開発版を試したい場合や、コードを改変して使用したい場合は、ソースコードからビルドします。

1. Rust のビルド環境がインストールされていることを確認します。
2. 以下のコマンドを実行してリポジトリをクローンします。
   `git clone https://github.com/adasima/opendaw.git`
3. クローンしたディレクトリに移動します。
   `cd opendaw`
4. 以下のコマンドでビルドと起動を行います。
   `cargo run --release`

## 詳細設定
現在、インストールや初回起動時に特別な設定ファイルは必要ありません。
設定はアプリケーション内で管理されます。

## 注意事項・制限
> - Linux 環境でソースコードからビルドする場合、`libasound2-dev` や `libudev-dev` などの依存ライブラリが事前にインストールされている必要があります。
