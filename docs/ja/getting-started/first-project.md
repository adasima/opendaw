# はじめてのプロジェクト

> OpenDAWを起動してから、簡単な楽曲を作成し、保存するまでの基本的な流れを解説するチュートリアルです。

## 概要
このページでは、OpenDAWの基本機能を使って、オーディオファイルを読み込み、MIDIノートを配置し、ミキシングを行ってプロジェクトを保存するまでのワークフローを体験します。

## 基本的な使い方

### 1. プロジェクトの準備
OpenDAWを起動すると、空のプロジェクトが自動的に開かれます。
デフォルトでは、BPMは **120.0** に設定されています。
画面下部のトランスポートコントロールから、テンポなどの基本設定を確認できます。

### 2. トラックの追加
楽曲を作成するには、まずトラックを追加する必要があります。
1. 画面左側のトラックパネルにある **+ Add Track** ボタンをクリックします。
2. 新しいトラックがリストに追加されます。

### 3. オーディオファイルのインポート
既存のドラムループなどのWAVファイルを読み込みます。
1. メイン画面にある **📁 Import Audio** ボタンをクリックします。
2. ファイル選択ダイアログから、手元の `.wav` ファイルを選択します。
3. ファイル名を持つ新しいトラックが自動的に追加されます。

### 4. MIDIノートの打ち込み（ピアノロール）
メロディやベースラインを作成するために、ピアノロールエディタを使用します。
1. MIDIデータを入力したいトラックを選択（※今後のアップデートで対応予定）またはMIDI用トラックとして意識します。
2. 画面中央の **Piano Roll** パネルで、グリッド上を「左クリック」してノートを追加します。
3. 間違えた場合は、ノートの上で「右クリック」して削除できます。

### 5. 再生とミキシング
作成したフレーズを再生しながら、各トラックのバランスを整えます。
1. 画面下部の **▶ (Play)** ボタンを押すか、`Space` キーを押して再生を開始します。
2. **Mixer & Effects** パネルで、各トラックの **Volume** スライダーをドラッグして音量を調整します。
3. 左右の広がりを持たせるために、**Pan** スライダーを調整します。
4. 全体の音量は **Master Volume** で調整します。

### 6. プロジェクトの保存
作成した状態を保存します。
1. 画面上部にある **💾 Save Project** ボタンをクリックします。
2. 保存先を指定し、`.aura` 形式のファイルとして保存します。
これで、後から **📂 Load Project** ボタンでいつでも作業を再開できます。

## 注意事項・制限
> - 現在、インポートしたオーディオファイルの実際の再生処理は開発中のため、トラックの追加のみが行われます。
