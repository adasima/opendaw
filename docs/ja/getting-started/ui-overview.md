# UIレイアウトの全体説明

> OpenDAW のメイン画面の各パネルの役割と配置について説明します。

## 概要
OpenDAW のメイン画面は、作業効率を最大化するために複数のパネルで構成されています。グラスモーフィズムデザインを採用し、直感的な操作が可能です。

## パネルの配置と役割
画面は大きく以下のパネルに分かれています。

1. **タイトルバー (Top)**
   - アプリケーション上部に配置されています。
   - **AI Agent** パネルの表示・非表示の切り替えや、**Settings** (設定) ドロップダウンからの言語切り替え機能を提供します。
2. **トラックパネル (Left)**
   - 画面左側に配置され、プロジェクト内のトラック一覧を表示します。トラックの追加や、**Session** / **Timeline** (セッションビューとタイムラインの切り替え)、**Plugins** (プラグインブラウザ) の切り替えが可能です。
3. **AI Agent & Track Details パネル (Right)**
   - 画面右側に配置されます。
   - **AI Agent**: AI アシスタントとのチャットインターフェースを提供します。
   - **Track Details**: トラックパネルでトラックを選択した際に表示され、ボリューム、パン、MIDIルーティングなどの詳細設定を行います。
4. **トランスポート & ミキサーパネル (Bottom)**
   - 画面下部に配置され、再生、停止、ループ、BPMの設定（トランスポートコントロール）と、マスターボリュームなどのミキサーコントロールを提供します。
5. **メインエリア (Center)**
   - 中央のメインエリアには、選択したモードに応じて以下のコンポーネントが配置されます。
     - **Timeline**: メインタイムラインおよびオーディオ・MIDIリージョンの表示（タイムラインキャンバス）。
     - **Session**: クリップランチャーを含むセッションビュー。
     - **Plugin Browser**: 外部プラグインのロードと一覧表示。
