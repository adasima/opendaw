# UIレイアウトの全体説明

> OpenDAW のメイン画面の各パネルの役割と配置について説明します。

## 概要
OpenDAW のメイン画面は、作業効率を最大化するために複数のパネルで構成されています。グラスモーフィズムデザインを採用し、直感的な操作が可能です。

## パネルの配置と役割
画面は大きく以下のパネルに分かれています。

1. **メニューバー (Top)**
   - プロジェクトの保存・読み込み（**Save Project** / **Load Project**）などのグローバルな操作を行います。
   - インポート（**Import Audio**）もここから行います。
   - **Arrangement** と **Session** ボタンを使用して、メインエリアの表示を切り替えます。
2. **トラックパネル (Left)**
   - 画面左側に配置され、プロジェクト内のトラック一覧を表示します。トラックの追加や削除が可能です。
3. **AI Agent パネル (Right)**
   - 画面右側に配置され、AI アシスタントとのチャットインターフェースを提供します。
4. **ミキサーパネル (Bottom)**
   - 画面下部に配置され、マスターボリュームや各トラックの音量、パン、エフェクトの調整を行います。
5. **メインエリア (Center)**
   - 中央のメインエリアには、上から順に以下のコンポーネントが配置されています。
     - **トランスポートコントロール**: 再生、停止、ループ、BPMの設定を行います。
     - **タイムライン & 波形表示**: 全体の再生位置（プレイヘッド）を表示します。
     - **ピアノロールエディタ**: MIDIノートの入力・編集を行います。
